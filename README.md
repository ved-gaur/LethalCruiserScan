# CruiserScan

A quality-of-life mod for Lethal Company.

Displays the **total scrap value** and **item count** currently stored on the **Company Cruiser** whenever you use the scanner.

---

## Features

- Shows **Cruiser Total** (combined scrap value on cruiser)
- Shows **Cruiser Items** (number of valid scrap items)

---

### Config

| Setting | Description |
|--------|-------------|
| `OffsetX` | Moves the display left/right |
| `OffsetY` | Moves the display up/down |
| `FontSize` | Changes text size |
| `DisplayDuration` | Seconds the display remains visible after scanning |
